# Chikara

A multithreaded, real time midi player using Vulkan.

# How-To Guide

## How to install

Not currently possible.

## How to play a MIDI

1. Open a terminal window
2. Go to the folder with your MIDI and Chikara
3. Run
```
Chikara.exe <MIDI FILE PATH HERE>
```
